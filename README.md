### English

# ghost-bits-analyzer


Ghost Bits (Cast Attack) Analyzer & Payload Fuzzer
Supports Unicode variant detection, underlying 8-bit semantic restoration, and random obfuscation of payload generation for testing vulnerability scenarios such as WAF bypass, path traversal, and protocol injection

This tool is based on the research presented at Black Hat ASIA 2026:https://i.blackhat.com/Asia-26/Presentations/Asia-26-Bai-Cast-Attack-Ghost-Bits-4.23.pdf

### 中文

一款针对 Java char (16-bit) 到 byte (8-bit) 截断风险（幽灵比特攻击）的渗透测试辅助工具。
支持 Unicode 变体探测、底层 8 位语义还原以及随机混淆 Payload 生成，用于测试 WAF 绕过、路径穿越及协议注入等漏洞场景 。

该工具基于在黑帽亚洲2026上展示的研究:https://i.blackhat.com/Asia-26/Presentations/Asia-26-Bai-Cast-Attack-Ghost-Bits-4.23.pdf
